#### PHP Framework
A DIY PHP Framework

#### Prerequisites
- PHP 8.1 or later
- `composer` for dependencies

#### Basic setup
Install composer dependencies
```
composer install
```

#### Run with the PHP built in web server
```
php -S localhost:8080 -t public public/index.php
```
